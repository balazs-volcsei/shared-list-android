# Shared List - Android

**Shared List** is a modern, efficient, and collaborative checklist application designed for Android. Built with modern Android tools, it allows users to manage tasks individually or collaborate with others via real-time synchronization.

## Key Features

- **Real-time Collaboration:** Share lists via email and sync changes instantly across all devices using Firebase.
- **Modern UI/UX:** Fully designed with **Material 3** and **Jetpack Compose** for a fluid, responsive experience.
- **Multi-language Support:** Native support for **English** and **Hungarian**.
- **Smart Monetization:**
    - Free tier with **AdMob** integration.
    - Rewarded video slots to unlock extra features for free.
    - Premium subscriptions and Lifetime access via **Google Play Billing**.
- **Offline First:** Reliable performance without an internet connection, with automatic syncing.
- **Security & Privacy:** Secure Google Sign-In and transparent data management.

## Tech Stack

- **UI:** Jetpack Compose with Material 3.
- **Architecture:** Clean Architecture with MVVM + Side Effects.
- **Dependency Injection:** Dagger Hilt.
- **Backend:** Firebase (Authentication, Firestore, Analytics).
- **Local Storage:** Room Database for caching & DataStore for settings.
- **Navigation:** Type-safe Navigation Compose.
